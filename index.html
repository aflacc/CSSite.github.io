A GitHub Project

This website is a website you can use as a template for your own websites. Powered by GitHub Pages and repl.it , Github Pages is a great place to start if you are planning on running your own website and Repl.it is a fantastic site to program with having practically every programming language. They're free, and efficient.

Want to copy this template for yourself? Click here to go to the downloads page, or scroll down to your prefered site theme.

This template comes with a .CSS file that contains a lot of formatting options such as italic , bold , Strikethrough and more formatting options.

Dark theme


The dark theme is prefered by most people. It is a lot easier on the persons eyes and is not blinding. Do you like this websites design? If you do, you might be a dark theme fan.
